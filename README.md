# Wadson Portfolio — 2026

Portfólio desenvolvido para Wadson Silva | Diretor de Arte & Designer Gráfico Sênior.

## 🚀 Tecnologias

- HTML5 / CSS3
- JavaScript (Vanilla)
- [Vite](https://vitejs.dev/)
- [Prettier](https://prettier.io/) / [ESLint](https://eslint.org/)

## 📂 Estrutura do Projeto

- `index.html`: Estrutura principal do site.
- `style.css`: Estilização completa e animações.
- `script.js`: Lógica de interatividade e animações.
- `public/`: Assets estáticos (imagens, vídeos, logos).
- `backup_referencias/`: Arquivos e referências externas.

## 🛠️ Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar a build de produção:
   ```bash
   npm run build
   ```
